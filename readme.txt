A WordPress plugin that creates a custom chatbot using the ChatGPT API with your own training material.

== Description ==

WP GPT Chatbot allows you to add an AI-powered chatbot to your WordPress website using the ChatGPT API. The chatbot is fully customizable and can be trained to respond based on your specific knowledge base or guidelines.

Features:
* Self-hosted solution with no reliance on third-party services
* Uses the OpenAI ChatGPT API
* Can be trained with your own materials to ensure accuracy and relevance
* Easy integration into any WordPress website
* Customizable appearance (colors, position, bot name)
* Mobile-friendly design

== Installation ==

1. Upload the `wp-gpt-chatbot` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the 'GPT Chatbot' menu in your WordPress admin area
4. Enter your OpenAI API key and configure the settings
5. Add your training material/instructions in the System Prompt field
6. Customize the appearance and save changes

== Frequently Asked Questions ==

= Do I need an OpenAI API key? =

Yes, you need to obtain an API key from OpenAI. You can sign up at https://platform.openai.com/

= How do I train the chatbot with my own material? =

Enter your training material/instructions in the System Prompt field in the plugin settings. This will guide the chatbot's responses.

= Can I customize the appearance of the chatbot? =

Yes, you can change the primary and secondary colors, position, bot name, and welcome message.
